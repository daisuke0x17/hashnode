もともと [Zenn](https://zenn.dev/daisuke23) を利用していました。特に不満があるわけでもないのですが移行します。  
強いて理由をあげるなら

* 自分で書いた記事を後から引き出しにくい（タグや文字列での検索が難しい）
    

でしょうか。  
Zenn で書いた記事も少しずつこちらへ移行しようと思います。Zenn 側は残したままにします。

移行先には [Hashnode](https://hashnode.com/) を選びました。選んだ理由としては

* 現時点で日本では流行ってない
    
* Web からの投稿を GitHub へバックアップ可能（Zenn と逆のフロー）
    
* タグや文字列での検索が可能
    

です。日本でまだ流行ってないことが自分にとっては大きいです。  
というのも、人気のプラットフォームで投稿するというのが自分にとって投稿の心理的ハードルを上げていました。

そしてこのブログの執筆に生成 AI は利用しません。

強制的に「自分の言葉でアウトプットする機会を設けたい」というのが一番の狙いです。  
扱うトピックは技術が中心になる予定です。正確な情報発信を心がけていますが、内容に誤りが含まれる可能性があります。お気づきの点があればご指摘いただけると幸いです。

本ブログの記事はすべて筆者個人の見解であり、所属する企業・組織の公式見解を代表するものではありません。
